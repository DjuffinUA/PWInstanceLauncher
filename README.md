# PWInstanceLauncher

## PW Instance Launcher — це локальний інструмент для зручного керування кількома ігровими вікнами клієнта. Додаток дозволяє зберігати профілі персонажів і запускати їх одним натисканням кнопки без необхідності щоразу вводити логін і пароль вручну.

## Програма спрощує роботу з кількома інстансами гри, надаючи можливість:
### * зберігати декілька профілів персонажів
### * швидко запускати клієнт для обраного персонажа
### * відкривати нові вікна гри в поточному або окремих віртуальних робочих столах

## Додаток працює повністю локально та не потребує підключення до інтернету для своєї роботи. Він не взаємодіє з зовнішніми сервісами та не передає жодних даних третім сторонам.
## PW Instance Launcher не замінює офіційний лаунчер гри та не втручається в його роботу. Це допоміжний інструмент для автоматизації запуску клієнта та підвищення зручності користування.
